# Smart Internship and Job Application Tracking System (SIJATS)

## Project Overview
SIJATS is an Information Systems Analysis and Design project developed to analyze and design a system for tracking internship and job applications in a structured and efficient way.

## Project Context
This project was prepared as part of the Information Systems Analysis and Design course.  
The focus of the project is on system requirements analysis, process modeling, and conceptual system design.

## Key Features
- Internship and job application tracking
- Structured data organization for applications
- User-centered system design approach
- Process and requirement analysis
